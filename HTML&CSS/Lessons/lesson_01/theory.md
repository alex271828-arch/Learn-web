<!--
  * <p></p> - будь-яка сторінка складається з будівельних блоків, ці блоки називають тегами (tag)

  ! Є два види тегів:
  - парні: для парних тегів обовʼязково повинен бути відкриваючий тег і закриваючий тег, а контент буде знаходитись між ними, наприклад: <p>Hello world!</p>
  - одиночні(самозакриваючі): у таких тегів немає контенту, тому їх не потрібно закривати, наприклад: <br> - тег для переносу рядка (з англ. break)

  ! Атрибути:
  у кожного тегу є спеціальні можливості або додаткові налаштування(для зміни поведінки або стилізації елементів). Всі атрибути пишуться всередині відкриваючого тегу, у вигляді name="value"
  Наприклад, я створю параграф і розміщу його по центру, для цього я використаю атрибут align
  <p align="center">Hello!</p>

  ! Структура документу:

  <!DOCTYPE html> - тег, який вказує тип документу і версію мови
  <html> - знову вказує тип документу і вся сторінка повинна знаходитись усередині цього тегу
    <head> - тег, у якому описується технічна інформація, важлива для браузера і коректної роботи сторінки
      <meta charset="UTF-8" /> - тег, який вказує кодування символів, щоб символи відображалися коректно на сторінці
      <title>My first Site</title> - заголовок для сайту на вкладці браузера
    </head>

    <body> - тіло сайту, тут ми будемо писати все, що хочемо показати користувачу на екрані

    </body>
  </html>

  ! Основні теги
  - <p></p> - тег для параграфів з текстом, у ньому просто пишеться текст, від англ. слова paragraph
  - <h1></h1>...<h6></h6> - 6 тегів для заголовків, чим більша цифра, тим менша приорітетність у заголовку. На сторінці може бути тільки один головний заголовок - <h1>. А всіх інших може бути скільки завгодно. Це як у книжок, назва може бути тільки одна, а кількість глав, розділів, підрозділів і тд може бути скільки завгодно.
  - <ul></ul> - маркерований(з крапочками) контейнер для списку
  - <ol></ol> - нумерований(з цифрами) контейнер для спику
  - <li></li> - один елемент списку
  - <a href=""></a> - створює гіпертекстові посилання, тобто це коли ти можеш посилання вставити всереди звичайного тексту. Для тегу а обовʼязково потрібно вказати атрибут href, всередину якого вставляється посилання (href - hyper reference - тобто, гіперпосилання). Ще є атрибут target="_blank", який вказує, що сайт відкриється у новій вкладці браузера
  - <b></b> - від англ. слова bold, тобто жирне начертання шрифту
  - <i></i>  від англ. слова italic, тобто курсивне начертання шрифту
  - <img src="" alt=""> - за допомогою цього тегу ми можемо створювати картинки, для цього потрібно написти два атрибути: src(source - ресурс, звідки взяти картинку), alt(alternative text - альтернативний текст) - якщо у користувача, наприклад, повільний інтернет, то картинки будуть завантажуватись повільно, або взагалі не завантажаться (або на сервері картинка пошкодилась), тому обовʼязково потрібно написати текст, який буде описувати те, що знаходиться на картинці, наприкдал, якщо на картинці букет квітів, то треба в цьому атрибуті написати квіти(або англ flowers)
  - <hr> - розділювальна лінія  
  - <div></div> - цей тег створений для того, щоб обʼєднувати спільні елементи у певні структурні контейнери
  - <header></header> - по суті, це звичайний div, але цей тег називається семантичним і існує для того щоб корекно розділювати секції. Наприклад цей тег створений для того, щоб зробити шапку сайту(саму верхню секцію)
  - <footer></footer> - по суті, це звичайний div, але цей тег називається семантичним і існує для того щоб корекно розділювати секції. Наприклад цей тег створений для того, щоб зробити підвал сайту (саму нижню секцію)
  - <main></main> - семантичний тег, який створений для головної секції на сайті
  - <section></section> - семантичний тег, який створений для того, щоб зробити звичайну секцію на сайті
  - <nav></nav> - по суті, це звичайний div, але цей тег називається семантичним і існує для того щоб створювати навігаційну панель по сайту (елементи меню)
 - <table></table> - контейнер для таблиці
 - <thead></thead> - контейнер для верхньої лінії таблиці
 - <tbody></tbody> - контейнер для контенту таблиці
 - <tfoot></tfoot> - контейнер для нижньої лінії таблиці
 - <tr></tr> - тег для створення однієї лінії в таблиці
 - <th></th> - тег для створення однієї комірки в таблиці (всередині тегу thead)
 - <td></td> - тег для створення однієї комірки в таблиці


! Основні атрибути

- width="300" - задає ширину елементу
